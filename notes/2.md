# Day 2

1. 安装了 zetachain cli

- npm install -g zetachain@latest
- 导入本地 eth 钱包：zetachain accounts import --type evm --private-key <私钥>
- 新建项目：zetachain new

2. 钱包连接 zetachain testnet

- 链 ID: 7001
- 链 rpc: https://zetachain-athens-evm.blockpi.network/v1/rpc/public
- faucet: https://www.zetachain.com/docs/reference/faucet
