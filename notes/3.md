# Day 3

1. 什么是 Universal App？

- Universal App 是 ZetaChain 上的智能合约，原生连接以太坊、BNB 和比特币等其他区块链。

2. 什么是 Gateway?

- 是一个接口，作为连接链上的合约与 ZetaChain 上的通用应用之间交互的统一入口。
- 分别部署在连接链和 Zeta 链，处理双方的合约调用逻辑。
